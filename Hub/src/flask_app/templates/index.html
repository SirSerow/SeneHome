Meteo

Веб-приложение для обработки, записи и отображения результатов измерения параметров окружающей среды.

Основные функции

  1. Приём данных от клиента
  2. Запись результатов измерений в базу данных RRD
  3. Демонстрация графиков на большом временном промежутке
  4. Отображение данных в реальном времени

Ссылки

  • Приложение принимает и обрабатывает данные в формате json. Адрес для отправки пакетов данных с клиента: 127.0.0.1:3000/meteo/register
  • Результаты измерений записываются в базу данных, а затем отображаются в виде графиков, доступных по адресу: 127.0.0.1:3000/meteo/graph
  • Отображение результатов измерений в реальном времени производится по адресу: 127.0.0.1:3000/meteo/realtime
  • Текущий статус соединения и конфигурации доступны по адресу: 127.0.0.1:3000/meteo/status
